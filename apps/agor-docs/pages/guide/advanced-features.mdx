# Advanced Features

Power-user features to enhance your AI coding workflow. An image is worth a thousand words, so let's show you what's possible.

## üéØ Worktree Highlighting

**Never miss when an agent finishes working.** Worktree cards pulse with a teal glow when sessions are ready for your attention.

![Worktree card with sessions](/screenshots/worktree_card_with_scheduled_sessions.png)

**What you'll see:**
- Pulsing teal glow around the entire card
- Teal border on ready sessions within the tree
- Animation automatically clears when you open the session

**Pro tip:** The glow only appears when the session is NOT already open in the conversation drawer.

## üîî Task Completion Chimes

**Hear when your agents finish working.** Set up audio notifications for long-running tasks so you can multitask.

![Audio settings](/Settings.gif)

**Quick setup:**
1. Open Settings ‚Üí Audio
2. Enable chimes and choose a sound (7 options)
3. Set volume (0-100%)
4. Set minimum duration (0-60 seconds)

**Available sounds:** Gentle Chime, Notification Bell, 8-Bit Coin, Retro Coin, Power-Up, You Got Mail, Success Tone

**Pro tip:** Set min duration to 30s to only hear chimes for long tasks. Click Preview to test!

## ‚öôÔ∏è User-Level Configuration

**Personalize Agor without affecting teammates.** Set your own API keys, environment variables, and agent defaults.

### API Keys

![User settings modal](/screenshots/settings_modal.png)

**Supports:** Anthropic, OpenAI, Gemini
**Security:** AES-256-GCM encrypted at rest
**Where:** Settings ‚Üí User Profile or Agentic Tools

**How it works:**
- Enter keys in masked inputs
- Green checkmark shows when key exists (without revealing value)
- Falls back to workspace env vars if not set
- Automatically injected into all your agent sessions

### Environment Variables

![Environment configuration](/screenshots/env_configuration.png)

**Use for:** Database URLs, service credentials, API tokens, custom paths

**Where:** Settings ‚Üí Environment Variables

### Agentic Tool Defaults

![Session creation with tool config](/screenshots/create_session_modal.png)

**Configure:**
- **Permission mode** - bypassPermissions, acceptEdits, plan, ask, auto, on-failure
- **Default model** - Sonnet 4.5, GPT-4o, Gemini Flash, etc.
- **MCP servers** - Which tools to include

**Where:** Create Session modal or Settings ‚Üí Agentic Tools

## üí° Power User Workflows

### Long Task Workflow
1. Set chime min duration ‚Üí 30s
2. Choose distinctive sound (Power-Up recommended)
3. Start task, switch to another window/worktree
4. Hear chime ‚Üí see pulsing glow ‚Üí review results

### Context Management Strategy
1. Watch context progression colors in conversations
2. Fork at 70-80% usage (orange)
3. Summarize learnings in forked session
4. Use summary as context for fresh exploration

### Multi-User Team Setup
1. Each user sets their own API keys
2. Configure personal env vars for dev databases
3. Set individual permission modes based on trust level
4. Work on shared boards without config conflicts

## üé® Visual Legend

### Worktree Card States
- **No border** - Idle, waiting
- **Pulsing teal glow** - Ready for your attention
- **Spinning icon** - Task actively running
- **Zone color border** - Pinned to a workflow zone

### Session States (in tree)
- **White border** - Normal/idle session
- **Teal border + background** - Ready for prompt
- **Fork icon** - Forked from parent session
- **Spawn icon** - Spawned subsession

### Context Window Colors
- **Green underlay** - Low usage (<50%)
- **Orange underlay** - Medium usage (50-80%)
- **Red underlay** - High usage (80-100%)
